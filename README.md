# Hoe start ik de website voor GameLink?
## Stap 1
Plaats de .env in de root van de website

## Stap 2
Open de terminal en doe 'npm i' of 'npm install'

## Stap 3
Voer in de terminal 'npm run dev'

## Stap 4
Ga naar de browser en voer in 'http://localhost:3000/'

## Stap 5
Ga in inspecteren en zet de grootte op 390 x 844 voor het goede schermformaat.
